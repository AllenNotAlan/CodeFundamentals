# CodeFundamentals

A personal repository for mastering software engineering fundamentals, algorithm implementations, and technical documentation.

## 🚀 Project Overview
This repository serves as a centralized hub for learning and implementing core computer science concepts using C# and .NET. It includes structured documentation for various algorithmic patterns and a practice environment for coding solutions.

## 🏗️ Technical Stack
- **Language:** C# 12.0
- **Runtime:** .NET 8.0 (Core Executable)
- **Environment:** Visual Studio / VS Code
- **Documentation:** Markdown (GitHub Flavored)

## 📁 Repository Structure
- **/CodeFundamentals**: The primary C# project containing algorithm implementations.
  - `Program.cs`: Main entry point for testing and executing code snippets.
- **/docs**: Comprehensive learning resources.
  - `/code-learning`: Detailed guides on patterns (Arrays, Hash Maps, Sliding Window, DP, etc.).
  - `/extras`: Placeholder for additional resources and advanced topics.
- **CodeFundamentals.sln**: Visual Studio solution file for multi-project management.

## 🧠 Patterns & Algorithms Tracked
1. Arrays & Strings
2. Hash Maps & Sets
3. Two Pointers
4. Sliding Window
5. Stacks & Queues
6. Sorting & Binary Search
7. Linked Lists
8. Trees & Recursion
9. Dynamic Programming

## 🛠️ Session Context (For Gemini CLI)
When starting a new session, this project should be treated as a **Learning & Implementation** environment.
- **Coding Style:** Idiomatic C# 12, focusing on performance and readability.
- **Workflow:** Implement algorithms in `Program.cs` and document findings in the relevant `/docs` file.
- **Build Command:** `dotnet build`
- **Run Command:** `dotnet run --project CodeFundamentals`
